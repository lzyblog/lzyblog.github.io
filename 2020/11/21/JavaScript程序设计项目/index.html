JavaSript教程

JavaScript快速入门

  • JavaScript是一种运行在浏览器中的解释型的编程语言。
  • 在Web世界里，只有JavaScript能跨平台、跨浏览器驱动网页，与用户交互.
  • JavaScript内嵌于HTML网页中,通过浏览器内置的JavaScript引擎直接编译，使静态转变成支持用户交互的页面程序。
语言 作用 说明
HTML 结构 从语义的角度，描述页面结构
CSS 样式 从审美的角度，美化页面
JavaScript 行为 从交互的角度，提升用户体验
  • JavaScript是脚本语言
    脚本(Script)就是一条条的文本命令，按照程序流程执行。常见脚本语言有 JavaScript,VBScript,Perl,PHP,Python等,而C,C++,Java,C#属于非脚本语言。
    非脚本语言需要编译、链接，生成独立的可执行文件后才能运行。
    脚本语言依赖于解释器，只在被调用时自动进行解释或调用。
  • JavaScript可以跨平台
    JavaScript语言不依赖操作系统，只需要浏览器的支持。
  • JavaScript支持面向对象

开发工具

  • JavaScript开发工具主要包括浏览器和代码编译器两种软件。
  • 浏览器用于执行、调试代码，代码编译器用于编写代码。
  • 种类繁多的浏览器只需要了解其内核就能对其有一个清晰的归类。
  • 浏览器内核分两部分：排版引擎和Javascript引擎。
  • sublime-text编译工具快捷键
    1. Ctrl + N，新建一个文档；
    2. Ctrl + Shift + P，打开命令模式，再输入 sshtml 进行模糊匹配，将语法切换到html模式；
    3. 输入 !，再按下Tab键或者Ctrl + E ，就能快速打开HTML5的整体结构。

引入方式

  • 嵌入式
    直接编写在HTML文件中

      <script type="text/javascript">
      JavaScript语句
  </script>

  • 外链式
    保存在一个单独的文件中，用“js”作为文件扩展名,最好放在body标签的底部，可以减少对整个页面的下载影响

      <script src="js/test.js"></script>

  • 行内式
    将Javascript代码作为HTML标签的属性值使用

      <a href="javascript:alert('Hello');">test</a>

JavaScript异步加载
无论使用内嵌式还是外链式，页面的下载和渲染都会暂停，等待脚本执行完成后才会继续。为了降低阻塞问题对页面造成的影响，可以使用可选属性：async和defer,添加后，即使文件下载失败，也不会阻塞javascript代码的执行

  • async
    async用于异步加载，即先下载文件，不阻塞其他代码执行，下载完成后再执行

      <script src="http://js.test/file.js" async></script>

  • defer
    defer用于延后执行，即先下载文件，直到网页加载完成后再执行

      <script src="http://js.test/file.js" defer></script>

数据类型

  • 布尔型

      var flag=true;

  • 数值型(Number)
    javaScript不区分整数和浮点数，统一用Number表示，以下都是合法的Number类型：

      123; // 整数123
  0.456; // 浮点数0.456
  1.2345e3; // 科学计数法表示1.2345x1000，等同于1234.5
  -99; // 负数
  NaN; // NaN表示Not a Number，当无法计算结果时用NaN表示
  Infinity; // Infinity表示无限大，当数值超过了JavaScript的Number所能表示的最大值时，就表示为Infinity

  • 字符型(String)
    字符串是以单引号’或双引号”括起来的任意文本

      var color='"red"blue';

  • 空型(Null)
    空型只有一个特殊的null值，用于表示一个不存在的或无效的对象或地址。

  • 未定义型(Undefined)
    未定义型只有一个特殊的undefined值，用于声明的变量还未被初始化时，变量的默认值为undefined。

undefined表示没有为变量设置值，null表示变量（对象或地址）不存在或无效。

  • 转布尔型
    Boolean()函数将如何非空字符串和非零的数组转为true，将空字符串、0、NaN、undefined和null转换为false
  • 转数值型
    Number()、parseInt( )、parseFloat()，isNaN()表示不是一个数值
  • 转字符型
    String()、toString()

函数

函数封装一段完成特定功能的代码。

    function 函数名([参数1, 参数2, ……])
    {
        函数体……
    }
    function maxNum(a, b) 
    {
          a =  parseInt(a);
         b =  parseInt(b);
          return a >= b ? a : b;
    }

  • 参数
    开发时若不能确定函数的形参个数，可以不设置形参；
    之后通过arguments对象获取调用时传递的实参。

      function sum(){
      console.log(arguments);
      var sum=0;
      for(var i in arguments){
          sum+=arguments[i];
      }
      return sum;
  }
  document.write(sum(1,2,3,4,5));

  • 变量的作用域
    全局变量：函数外声明，或者函数内省略var声明；
    局部变量：函数体内用var声明。
    块级变量：ES6提供的let关键字声明，仅在”{}”内有效。

  • 匿名函数
    匿名函数指没有函数名称的函数，可以避免全局变量的污染以及函数名的冲突问题。

  • 回调函数
    回调函数指一个函数A作为参数传递给一个函数B，然后在B的函数体内调用函数A，称A为回调函数。其中，匿名函数常作为函数的参数传递，实现回调函数。

  • 闭包函数
    JavaScript中内部函数可引用外部函数的变量，但是在函数外部则不能访问函数的内部变量和嵌套函数，此时就需要使用”闭包“。
    ”闭包“指有权访问另一函数作用域内变量(局部变量)的函数。它主要的用途是以下两点：

  1. 可以在函数外部读取函数内部的变量
  2. 可以让变量的值始终保持在内存中

JSON

  • JSON: JavaScript Object Notation(JavaScript 对象表示法)。

  • JSON 是轻量级的文本数据交换格式，类似 XML，但 比 XML 更小、更快，更易解析。

  • JSON 独立于语言和平台，目前非常多的动态（PHP，JSP，.NET）编程语言都支持JSON。

  1. JSON对象

     //JSON对象
 var obj = {“name”:“Jerry”，“age”:15} 
 //JS对象
 var obj = {name:“Jerry”，age:15，    
          sayHello:function{……}
          }

    • JSON对象的属性名（key）必须被包含在双引号之中，而JavaScript对象除了有空格的属性名、中间有连字符-的属性名必须在双引号之中外，其它随意；

    • 不能在JSON对象中定义方法，而在JavaScript对象中可以；

    • JSON对象可以被很多语言操作，而JavaScript对象只有JS自己可以识别。

  • 事件：用户行为，如单击、鼠标移动等；
  • 事件处理程序：对事件进行处理的代码；
  • 事件驱动式：侦测到用户行为、执行事件处理程序的过程；
  • 事件流：事件传播的过程（捕获+冒泡）
    1. 事件捕获方式（网景）

    2. 事件冒泡方式（微软）
      事件流传播的顺序应该是从发生事件的元素节点到DOM树的根节点。

正则表达式

正则表达式查询网站I Hate Regex

  1. 什么是正则表达式
    正则表达式（Regular Expression，简称regexp）

    • 概念：是一种描述字符串结构的语法规则（描述字符串特征）
    • 作用：用于验证各种字符串是否匹配这个特征。
    • 应用：在项目开发中，手机号码指定位数的隐藏、数据采集、敏感词的过滤以及表单的验证等功能，都可以利用正则表达式来实现。

  2. 获取正则对象

    • 字面量方式
        /pattern/flags
  例如：/abc/gi
    • RegExp对象构造函数方式
        new RegExp(pattern[,flags])
  RegExp(pattern[,flags])
  //示例
  var str='Abc123abc456abc';
  var reg=new RegExp('abc','gi');
  console.log(str.match(reg));
      pattern：模式文本，由元字符和文本字符（普通字符）组成。
      flags：模式修饰符。
      模式符 说明
      g 用于在目标字符串中实现全局匹配（多次）
      i 忽略大小写
      m 实现多行匹配
      u 以Unicode编码执行正则表达式
      y 粘性匹配，仅匹配目标字符串中此正则表达式的lastindex属性指示的索引
      ^ 匹配字符串开始的位置
      $ 匹配字符串结束的位置

  3. 使用正则对象

    • RegExp对象提供的exec()方法，一次仅返回一个匹配结果，匹配成功返回数组，否则返回null
        var str='Abc123abc456abc';
  var reg=/abc/gi;
  console.log(reg.exec(str)); //返回Abc
    • String对象提供的match()方法，匹配出所有符合要求的内容，匹配成功返回数组，否则返回false
        var str='Abc123abc456abc';
  var reg=/abc/gi;
  console.log(str.match(reg)); //返回Abc、abc、abc

  4. 转义字符的处理
    使用反斜线对特殊字符进行转义

     var str='^abc\\1.23*edf$';
 var reg1=/\./;
 console.log(str.match(reg1));
 var reg2=new RegExp('\\.');
 console.log(str.match(reg2));

字符类别与集合

  • 字符类别
    使正则表达式更加简洁，便于阅读。
字符 含义 字符 含义
. 匹配除“\n”外的任何单个字符 \f 匹配一个换页符（form-feed)
\d 匹配任意一个阿拉伯数字 \D 匹配任意一个非阿拉伯数字字符
\s 匹配一个空白符，包括空格、制表符、换页符、换行符等 \S 匹配一个非空白符
\w 匹配任意一个字母（大小写）、数字和下划线 \W 匹配任意一个非“字母（大小写）、数字和下划线”的字符
  • 字符集合
    正则表达式中的”[]”可以实现一个字符集合
pattern 说明
[cat] 匹配字符集合中的任意一个字符c、a、t
[^cat] 匹配除c、a、t以外的字符
[B-Z] 匹配字母B~Z范围内的字符
[^a-z] 匹配字母a~z范围外的字符
[a-zA-Z0-9] 匹配大小写字母和0~9范围内的字符
[\u4e00-\u9fa5] 匹配任意一个中文字符

字符限定与分组

  • 限定符
      var reg=/\d{6}/gi; //匹配6个连续出现的数字
限定符 说明 举例
? 匹配前面的字符零次或一次 ht?t
+ 匹配前面的字符一次或多次 bre+ad
* 匹配前面的字符零次或多次 ro*se
{n} 匹配前面的字符n次 hit{2}er
{n,} 匹配前面的字符最少n次 hit{2,}er
{n,m} 匹配前面的字符最少n次，最多m次 fe{0,2}

  • 贪婪与懒惰匹配

    1. 贪婪匹配表示匹配尽可能多的字符

    2. 懒惰匹配表示匹配尽可能少的字符

    3. 在默认情况下，是贪婪匹配，若想实现懒惰匹配，需在上一个限定符后面加上”?”符号

        var str='webWEBWebwEb';
  var reg1=/w.*b/gi;   //贪婪匹配
  var reg2=/w.*?b/gi;     //懒惰匹配
  console.log(reg1.exec(str));   //webWEBWebwEb 获得最先出现的w到最后出现的b
  console.log(reg2.exec(str));   //web 获得最先出现的w到最先出现的b

括号字符

被括号字符“()”括起来的内容，称之为”子表达式“

  1. 改变限定符的作用范围
     //改变前
 正则表达式：catch|er
 可匹配的结果：catch、er
 //改变后
 正则表达式：cat(ch|er)
 可匹配的结果：catch、cater
  2. 分组
     //分组前
 正则表达式：abc{2}
 可匹配的结果：abcc
 //分组后
 正则表达式： a(bc){2}
 可匹配的结果：abcbc
  3. 捕获
    将子表达式匹配到的内容存储到系统的缓存区中。
     var res1 = '1234'.match(/\d\d\d\d/g);
 console.log(res1); //["1234"]
 var res2 = '1234'.match(/(\d)(\d)(\d)(\d)/);
 console.log(res2); // ["1234","1","2","3","4"]
 var res3 = '1234'.match(/(\d)(\d)(\d)(\d)/g);
 console.log(res3);  //["1234"]
  4. 反向引用
    在正则表达式中，获取存放在缓存区内的子表达式的捕获内容，则可以使用”\n“(n>0)的方式引用
     \1：第一个子表达式捕获的内容
 \2：第二个子表达式捕获的内容
 var reg1=/(\d)\1\1/gi; 连续三个相同的数字
 var reg2=/(\s)(\s)\1\2/gi; ABAB格式

正则相关方法

  1. RegExp对象 的test()方法：
    检测正则表达式与指定的字符串是否匹配，返回true或false。
     var reg = /([A-Z])([A-Z])\1\2/g;
 console.log(reg.test('1234'));    //false    
 console.log(reg.test('abab'));    //false
 console.log(reg.test('CDCD'));    //true
 console.log(reg.test('EfEf'));    //false
  2. String对象的search()方法：
    可以返回指定模式的子串在字符串首次出现的位置，查找失败返回-1。
     var str = '123*abc.456';
 console.log(str.search('.*'));       //0    
 console.log(str.search(/[\.\*]/));   //3

    search()方法如果传入一个非正则表达式对象，则会将其转换为正则表达式对象

  3. String对象的split()方法
    用于根据指定的分隔符将一个字符串分割成字符串数组，其分割后的字符串数组中不包括分隔符。
    当分隔符不只一个时，需要定义正则对象才能够完成字符串的分割操作。
     var str = 'test@123.com';
 var reg = /[@\.]/;
 var split_res = str.split(reg);
 console.log(split_res);    // 输出结果：(3) ["test", "123", "com"]
  4. String对象的replace()方法
    用 newChar 字符替换字符串中出现的所有 searchChar 字符，并返回替换后的新字符串。
     var str = 'test@123.com';
 var str_new= str.replace(‘123’,’163’);
 console.log(str_new);    // 输出结果：test@163.com

replace：获取捕获内容，并对捕获内容进行替换

    var str='12ab34';
    var reg=/(\d)(\d)/gi;
    console.log(str.match(reg));
    console.log(str.replace(reg,'*'));  //["12",["34"]]  *ab*

$n:获取第n个字表达式捕获的内容

    var str = 'Regular Capture';
    var reg = /(\w+)\s(\w+)/gi;
    var newStr = str.replace(reg,'$2 $1');
    console.log(newStr);
