Hexo+GitHub搭建个人博客

一.简介

一、什么是GitHub？

GitHub Pages 是由 GitHub 官方提供的一种免费的静态站点托管服务，让我们可以在 GitHub 仓库里托管和发布自己的静态网站页面。

About GitHub Pages

2.什么是Hexo？

Hexo 是一个快速、简洁且高效的静态博客框架，它基于 Node.js 运行，可以将我们撰写的 Markdown 文档解析渲染成静态的 HTML 网页。

Hexo官网

3.Hexo+GitHub文章发布原理

在本地撰写 Markdown 格式文章后，通过 Hexo 解析文档，渲染生成具有主题样式的 HTML 静态网页，再推送到 GitHub 上完成博文的发布。

二、准备

  • 输入代码时，核对准确，最好切换成英文输入法；
  • 将文中的 “用户名” 和 “邮箱” 替换为自己的 GitHub 账户名和绑定的邮箱；
  • 统一使用 Git Bash 或者cmd进行操作（支持 Win、Mac）；

三、环境搭建

Hexo 基于 Node.js，搭建过程中还需要使用 npm（Node.js 已带） 和 git，因此先搭建本地操作环境，安装 Node.js 和 Git。

Node.js下载网址
Node.js安装教程

Git下载地址
Git安装及配置
安装完成后，Win+R 输入 cmd 并打开，依次输入(注意空格)

node -v
npm -v
git --version

如果出现”npm”不是内部指定，说明环境变量没配好,Node.js文件目录复制路径到Path中即可.
回车，出现程序版本号即可。

四、安装Hexo

打开cmd,输入以下命令
1.通过淘宝镜像安装博客框架
2.如果出现”cnpm”不是内部指定，说明环境变量没配好,根据提示目录复制路径到Path中即可.

npm install -g cnpm --registry=https://registry.npm.taobao.org
cnpm -v
cnpm install -g hexo-cli
hexo -v

五、Hexo搭建

1.常用cmd操作:

返回根目录:cd/
进入某目录:cd 目录名
返回上一目录:cd ..
查看文件目录:dir

2.创建文件夹存放hexo程序文件

mkdir 文件名

3.根据命令操作(文件名以C:/blog/lzyblog为例，最后使用父目录创建子目录来使用)

进入lzybolg:
cd blog
cd lzyblog
初始化Hexo,需要等待一会:   hexo init
启动Hexo,可以通过本地访问:  hexo s
可以通过lzyblog/source/_posts/xxxx.md 中修改博客
修改后: hexo clean
生成Hexo程序: hexo g
启动Hexo，本地预览: hexo s

六、部署到GitHub

1.注册登录GuiHub

注意用户名与网址访问域名要一致

2.创建GuiHub库(new repository)

注意Repository name：用户名.guihub.io (格式不能变)

3.安装GuiHub部署插件

cmd输入：cnpm install --save hexo-deployer-git

4.设置_config.yml文件

可以使用笔记本编辑，推荐使用sublime text

_config.yml文件修改(最下面):
 type: git
 repo:  https://github.com/lzyblog/lzyblog.github.io （仓库url）
 branch: master 

cmd命令输入:
git config --global user.name("登录用户名")
git config --global user.email("登录邮箱")
提交到GitHub: hexo d

5.修改博客

通过C:\blog\lzyblog\source\_posts目录下xxx.md文件修改
(如果发现peployer不存在,需要重新引入部署插件)：cnpm install --save hexo-deployer-git
hexo s （可在本地预览） 
hexo g  (部署)
hexo d   (提交Github)
输入网址:lzyblog.guihub.io

七.修改主题

登录Hexo网址

git clone 主题url.git themes/主题名
然后_config.yml配置:theme: 主题名
hexo clean
hexo g
hexo s (预览)
hexo d

Hexo更改主题后启动服务器，界面显如下字符:

extends includes/layout.pug block content include includes/recent-posts.pug include includes/partial

解决方案

npm install --save hexo-renderer-jade hexo-generator-feed hexo-generator-sitemap hexo-browsersync hexo-generator-archive

清除缓存

hexo clean

生成静态文件即可

hexo g

八.Markdown标记语言

1.简介

Markdown 是一种轻量级标记语言，它允许人们使用易读易写的纯文本格式编写文档。

Markdown 语言在 2004 由约翰·格鲁伯（英语：John Gruber）创建。

Markdown 编写的文档可以导出 HTML 、Word、图像、PDF、Epub 等多种格式的文档。

Markdown 编写的文档后缀为 .md, .markdown。
