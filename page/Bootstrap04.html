过渡效果

模态框

  • modal.js插件，千万不要在一个模态框上重叠另一个模态框。不要在一个页面打开多个模态框
  • 务必将模态框的 HTML 代码放在文档的最高层级内（也就是说，尽量作为 body 标签的直接子元素），以避免其他组件影响模态框的展现和/或功能

触发模态框

  • 第一种方法是不需写 JavaScript 代码也可激活模态框。通过在一个起控制器作用的元素（例如：按钮）上添加 data-toggle="modal" 属性，或者 data-target="#foo" 属性，再或者 href="#foo" 属性，用于指向被控制的模态框
  • 第二种是使用JavaScript 代码，通过元素的 id myModal 调用模态框，相关属性参考官网
