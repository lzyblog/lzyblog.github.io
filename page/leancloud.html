Butterfly主题配置Valine评论

创建LeanCloud应用

  • 进行LeanCLoud账号注册，官网：https://console.leancloud.app

注意：一定要选择国际版，如果你选择的是华北或者华东的话，Valine 后台评论管理是需要备案才能绑定的，但本人注册的是华北版，但演示步骤都是一样的

  • 登录LeanCloud后，点击创建应用填写应用昵称点击创建即可

获取应用Key

  • 打开应用选择设置–>应用Keys–>复制AppID和AppKey,找到Hexo配置文件中valine配置即可

Hexo配置

  • 打开主题配置文件config.yml，找到valine配置项

邮件提醒功能

自定义环境变量

  • 打开LeanCloud的控制台，然后进入应用的设置界面，然后通过【云引擎 | 设置】菜单，进入到云引擎的相关配置面板，然后在”自定义环境变量“处新建以下变量名和值：

变量名 示例 说明
SITE_NAME 小李博客 [必填] 网站名称
SITE_URL https://xiaoliblog.cn [必填] 网站地址，最后不要加 /
SMTP_USER 2312057536@qq.com [必填]一般为邮箱地址
SMTP_PASS xxxxx [必填] SMTP 密码，邮箱授权码
SMTP_SERVICE QQ 邮件服务提供商，👉 内置支持
SENDER_NAME 小李博客 [必填] 寄件人名称
TEMPLATE_NAME rainbow [必填] 设置提醒邮件的主题 rainbow/custom1/custom2
ADMIN_URL https://xxxx.leanapp.cn/ [可选] 后台管理地址 (非博客地址)
SCKEY SCU100890xxxxxx293e1xxx9dbfe82axxxx2dc [可选] server 酱的 SCKEY(微信提醒)
QMSG_KEY fdexxxx42a1xxx28f16d5xxxxa00 [可选] Qmsg 酱的密钥
AKISMET_KEY xxxxxxxxxxxx [可选] Akismet Key 用于垃圾评论检测
QQ 2312057536 [可选] Qmsg 酱发送的 qq，不填为全部。支持多个，用英文逗号分隔即可
QQ_SHAKE true [可选]，填写代表发送 QQ 戳一戳
INFO <p style=’color:red’>test<p> [可选] 自定义信息输出，支持HTML代码LeanColud控制台日志输出
favicon 图片URL [可选] 网页 favicon 图标
TO_EMAIL 2312057536@qq.com [可选] 博主通知收件地址，默认使用 SMTP_USER
SPAM_WORDS 检测的内容，如：单号，物流 [可选] 需要对屏蔽的关键词，关键词用半角逗号分隔
MAIN_COLOR #ff9191 [可选] 仅针对 custom2 模板主题的主要颜色
AKISMET_KEY xxxx [可选] Akismet Key 用于垃圾评论检测，设为 MANUAL_REVIEW 开启人工审核，留空不使用反垃圾
COMMENT #post-comment [可选] 评论 div 的 ID 名

每一次更改都需要重新部署生产环境

部署生产环境

  • 接下来部署生产环境。依次点击【云引擎 | 部署】菜单，打开部署界面，然后通过如下所示部署Valine-Admin，如下：


以上的GitHub地址如下”https://github.com/DesertsP/Valine-Admin.git" ，必须配置这个地址，不然点击部署按钮后，它在部署过程中会报错。

创建定时任务

  • 部署成功后，我们设置定时任务，因为免费的LeanCloud会有自动休眠模式，所以要设置定时任务然它自动重启，操作过程如下：

名称 函数 Cron表达式
定时检查24小时内漏发的邮件 resend_mails 0 0 8 * * ?（定时检查24小时内漏发的邮件通知）
自动唤醒 self_wake 0 0/30 7-23 * * ? （表示每天早7点到晚23点每隔30分钟访问云引擎）

设置安全域名

  • 定时任务设置完成后，我们为云引擎绑定一个域名，为了保证后续邮件提醒功能正常，博客有多少个地址就填多少个地址，不然评论显示403，每一个域名占一行

QQ/微信提醒

Service 酱 SCKEY 获取

  • 首先需要到 server 的官网，开启微信提醒并获取 SCKEY。
  • 注册—>登陆—>发送消息—>就可以得到SCKEY代码了，具体步骤请戳Server酱官网，里面写的非常详细了
  • 开启微信通知后可以在官方那里测试下能否接到通知（关注一个微信公众号）
  • 复制SCKEY复制到LeanCloud的SCKEY环境变量后保存即可

Qmsg 酱key密钥获取

Qmsg 的官网为：https://qmsg.zendee.cn/, 进入官网，登录账号（qq 或者神邸用户都可以

  • 选择并添加一个消息发送者，并填写你接受消息的 qq 号。（可以使用小号登录，大号接收）

  • 获取密钥
    点击右上角的文档，把send后面的key复制到LeanCloud的QMSG_KEY环境变量后保存即可
