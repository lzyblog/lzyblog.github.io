C语言⭐结构体与共用体

发表于2021-01-15|更新于2021-01-24|编程语言C
|字数总计:2.1k|阅读时长:9分钟|阅读量:

结构体

将不同类型的数组组合成一个有机的整体，以便于引用。

结构体定义

定义一个结构的一般形式为：

    struct 结构名
    {
        成员列表
    };

成员列表由若干个成员组成，每个成员都是该结构的一个组成部分。对对每个成员也必须作类型说明，其形式为:

    类型说明符 成员名;
  • 定义结构类型变量的方法

①先声明结构体类型再定义变量名

    类型名      结构体   变量名1，变量名2
    struct student student1,student2;

定义了student1和student2为strcut student类型的变量,即它们具有strcut student类型的结构。

②在声明类型的同时定义变量

    strcut 结构体名
    {
        成员表列
    }变量名变量;

    strcut student
    {
        int num;
        char name[20];
        char sex;
        int age;
        float score;
        char addr[30];
    }student1,student2

在定义了结构体变量后，系统会为之分配内存单元
例如：student1和student2在内存中各占(4+20+1+4+4+30=67)个字节。

③直接定义结构体类型变量

    struct
    {
        成员表列
    }变量名表列;

即不出现结构体名。

  • 结构体嵌套

结构体变量的引用

不能将一个结构体变量作为一个整体进行输出
正确引用方式：

    结构体变量名.成员名
    student1.num  //表示student1变量中的num成员
    student1.num=100; //对变量成员赋值

  • 如果成员本身又属于一个结构体类型，则要用若干个成员运算符，一级一级地找到最低的一级的成员。只能对最低级的成员进行赋值或存取以及运算。
    例如以上结构体嵌套：

      student1.num;
  student1.birthday.month  //多个点运算符访问最低级成员

  • 对结构体变量的成员可以向普通变量一样进行各种运算(根据其类型决定可以进行的运算)

  • 可以引用结构体变量成员的地址，也可以引用结构体变量的地址。
    结构体变量的地址注意用在函数参数，传递结构体变量的地址。

结构体变量初始化

结构体数组

结构体数组每一个元素都是一个结构体类型的数据，它们都分别包括各个成员项。

结构体数组定义

    //第一种方法
    struct student
    {
        int num;
        char sex;
    };
    struct student a[3];  //a为结构体数组名
    //第二种方法
    struct student
    {
        int num;
        char sex;
    }a[3];

结构体数组初始化

    //第一种方法
    struct student
    {
        int num;
        char sex;
    }a[3]={{1,'M'},{2,'M'},{3,'M'}};
    //第二种方法
    struct student
    {
        int num;
        char sex;
    };
    struct student a[3]={{1,'M'},{2,'M'},{3,'M'}};

指向结构体类型数据的指针

  • 一个结构体变量的指针就是该结构体变量所占据的内存段的起始地址。

  • 可以设一个指针变量，用来指向一个结构体变量，此时该指针变量的值是结构体变量的起始地址。

  • 指针变量也可以用来指向结构体数组中的元素。

  • 结构体变量声明的一般形式:

      struct 结构名 *结构指针变量名
  struct student *pstu;

  • 赋值是把结构变量的首地址赋予该指针变量，不能把结构名赋予该指针变量。因为结构名只能表示一个结构形式，编译系统并不对它分配内存空间。

  • 有了结构指针变量，就能更方便地访问结构变量的各个成员。
    其访问形式如下:

      (*结构指针变量).成员名
  (*pstu).num;
  //或者
  结构指针变量->成员名
  pstu->num

结构指针变量作函数参数

将一个结构体变量的值传递给另一个函数，有3个方法:

  • 用结构体变量的成员作参数
  • 用结构体变量作实参
  • 用指向结构体变量(或数组)的指针作实参，将结构体变量(或数组)的地址传给形参

共用体

使几个不同的变量共同占同一段内存的结构称为“共用体”类型的结构。
定义共同体类型变量的一般形式为:

    union 共用体名
    {
        成员表列
    }变量表列;

例如

    union data
    {
        int;
        char ch;
        float f;
    }a,b,c;
    //或
    union data
    {
        int i;
        char ch;
        float f;
    };
    union data a,b,c;
