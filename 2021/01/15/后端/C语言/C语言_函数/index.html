C语言⭐函数

发表于2021-01-15|更新于2021-01-15|编程语言C
|字数总计:3.3k|阅读时长:12分钟|阅读量:

函数

  • 无参函数
      类型名 函数名()
  {
      //函数体包括声明部分和语句部分
  }
  • 有参函数
      类型名 函数名(形式参数列表)
  {
      //函数体
  }

    虚实结合

      int max(int x,int y); //x，y为形参
  int a=1,b=2,m;
  m=max(1,2);   //a,b为实参
    形参并不占内存中的存储单元，当被调用时，临时分配内存单元。
    实参向形参的数据传递是”值传递’，单向传递，形参不能传给实参，因为他们具有不同的存储单元。

返回值

函数的返回值是通过return语句获得的。
函数类型决定返回值的类型。

    int max(int x,int y)
    {
    return(x>y?x:y);
    }

对于不带返回值，可以定义函数为void，定义后不允许出现return语句。

函数声明

用户自定义函数，需要在主调函数中对被调函数做声明。其作用是把函数的具体信息(函数名、函数参数个数，参数类型等)通知给编译系统。

函数的递归调用

  • 函数不可以嵌套定义，但可以嵌套调用。

  • 当调用一个函数的过程中又出现直接或间接地调用该函数本身，称为函数的递归调用。

  • 通过使用if语句来终止无止尽的递归。

  • 一个递归的问题可以分为”回溯”和”递推”两部分。

数组作为函数参数

  • 数组元素可以作为函数实参，不能用作形参。(因为形参是函数被调用时临时分配内存，不可能为单个数组元素分配存储单元，数组都是连续存放的）

  • 数组名可以作为函数实参或者形参。注意，数组名做实参时，向形参传递的是数组元素首地址。

  • 当形参数组名获得了实参数组的首地址后，它们共占同一存储单元，具有相同的值。

局部变量和全局变量

  • 在一个函数内定义的变量是局部变量，只在本函数范围内有效。

  • 在函数外定义的变量是全局变量，可以为本文件中其他函数所共用。

全局变量在程序的全部执行过程中都占用存储单元。

变量的存储方式和生存期

从变量的作用域来看，变量可以分全局变量和局部变量。
从变量生存期来看，变量的存储可以分静态存储方式和动态存储方式。

  • 静态存储方式：指在程序运行期间由系统分配固定的存储空间的方式。
  • 动态存储方式：指在程序运行期间根据需要进行动态的分配存储空间的方式。

内存中共用户使用的存储空间分3部分：
①程序区
②静态存储区
③动态存储区
数据分别存放在静态存储区和动态存储区中，全局变量全部存放在静态存储区中，在程序开始执行时给全局变量分配存储区，程序执行完毕后释放，在执行过程中占据固定的存储空间。
在动态存储区中存放以下数据：
①函数形式参数
②函数中没有用关键字static声明的变量，即自动变量
③函数调用时的现场保护和返回地址等

存储类别

每一个变量和函数都有两个属性：数据类型和数据的存储类别（如静态存储和动态存储）。
C的存储类别包括4种：自动的(auto)、静态的(static)、寄存器的(register)、外部的(extern)。

  • 自动变量auto
    自动变量动态分配存储空间，属于动态存储类别，数据存储在动态存储区中，函数调用结束后即释放，函数中的形参和定义的局部变量都属于此类。

实际上关键字”auto“可以省略，不写auto则隐含指定为”自动存储类别“。

  • 静态局部变量static
    静态局部变量属于静态存储类别，在静态存储区内分配存储单元，在程序整个运行期间都不释放。

    对静态局部变量是在编译时赋初值的，即只赋值一次，以后每次调用函数时不用重新赋值而是保留上次函数调用结束时结束的值。

  • 寄存器变量register
    变量的值一般是存放在内存中的，当程序用到哪一个变量的值是，由控制器发出指令将内存中该变量的值送到运算器进行运算，如果需要存数，再从运算器将数据送到内存中。
    如果有一些变量使用频繁，为提高效率，允许将局部变量的值放到CPU中的寄存器中，这种变量叫寄存器变量。

      register int f;

外部变量extern

此类属于全局变量范围，全局变量作用域从变量的定义处开始，到本程序文件的末尾。并通过extern可以扩展变量的作用域。

  • 在一个文件内扩展外部变量的作用域
    可以是把作用域扩展到变量定义处之前。
  • 将外部变量的作用域扩展到其他文件
    如果一个程序包括两个文件，两个文件都需要用到同一个外部变量Num，则可以在任一个文件中定义外部变量Num，在另一个文件中用extern对Num做外部声明，即”extern Num“。
  • 将外部变量的作用域限制在本文件中
    static声明可以将外部变量只限于本文件引用。

内部函数和外部函数

  • 内部函数
    如果一个函数只能被本文件内其他函数所调用，则称为内部函数（静态函数）。

      static 类型名 函数名（形参名）
  static int fun(int a,intb);

  • 外部函数
    如果定义函数时省略extern，则默认为外部函数。

      extern int fun(int a,int b);

enter_string为函数原型，函数原型能够把函数的作用域扩展到定义该函数的文件之外（不需要extern),典型例子是#include指令的作用。
